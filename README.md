### How to use

**Step 1:**

Download or clone this repo:

```bash
git clone https://github.com/ChaohaoDu/Flutter-Budget-App.git
```

**Step 2:**

Go to project root and execute the following command in console to get the required dependencies:

```bash
flutter pub get 
```

**Step 3:**

Run the app

```bash
flutter run
```
